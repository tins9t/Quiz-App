This is our Quiz app Quizilla. That has a websocket backend and flutter frontend.

To run backend go to the api folder and run the following commands:

dotnet run

To run frontend go to the frontend folder and run the following commands:

flutter run

This will start the frontend and backend. You can now start using the app.
